# 帳戶管理

「帳戶」代表現實中你的一個資金容器——銀行戶頭、信用卡、現金錢包、券商戶頭等。每一筆交易都會歸屬到某一個帳戶。

## 欄位說明

| 欄位 | 說明 | 範例 |
| ---- | ---- | ---- |
| `name` | 顯示用名稱 | 「玉山活存」、「國泰世華信用卡」 |
| `type` | 帳戶類型（5 選 1） | `SAVING` |
| `currency` | 幣別代碼（3 字 ISO 4217） | `TWD`、`USD`、`JPY` |
| `currentBalance` | 當下餘額 | `50000.00` |

### 帳戶類型 (type)

| 值 | 用途 |
| -- | ---- |
| `CHECKING` | 支票 / 活期 |
| `SAVING` | 儲蓄存款 |
| `CREDIT` | 信用卡（餘額代表「未繳金額」，是負債）|
| `INVESTMENT` | 投資戶頭（券商、基金）|
| `CASH` | 現金 |

!!! note "未在清單中的類型怎麼辦？"
    短期內可以挑最接近的；長期會視需求新增類型。

## 操作清單

| 操作 | API | 說明 |
| ---- | --- | ---- |
| 列出我的所有帳戶 | `GET /api/v1/accounts` | 依 id 升序 |
| 取得單一帳戶 | `GET /api/v1/accounts/{id}` | 取不到回 404 |
| 新增帳戶 | `POST /api/v1/accounts` | 成功回 201 |
| 修改帳戶 | `PUT /api/v1/accounts/{id}` | 修改 name/type/currency/balance |
| 刪除帳戶 | `DELETE /api/v1/accounts/{id}` | 成功回 204 |

完整 request/response 見 [帳戶 API](../api-reference/accounts.md)。

## 規則與限制

- **隔離**：你只看得到自己的帳戶；別人查你的帳戶 id 會回 `404`（不是 403，避免洩漏存在性）。
- **餘額**：可以是負數（如信用卡）；小數點 2 位。
- **幣別**：目前**不會自動換算**——不同幣別的帳戶就分開看。多幣別與匯率將在 Sprint 5 處理。
- **刪除**：尚未連動「交易」。Sprint 2 加入交易後，刪除帳戶會先擋掉（避免孤兒交易）。

## 常見問題

**Q：我可以幫每個信用卡開一個帳戶嗎？**
可以，建議**一張卡一個帳戶**，方便對帳。

**Q：薪轉戶要算 `CHECKING` 還是 `SAVING`？**
台灣銀行幾乎都是綜合存款，算 `SAVING` 即可。

**Q：投資帳戶的持倉怎麼辦？**
Sprint 5 後會自動估值；目前先手動把市值填到 `currentBalance`。
