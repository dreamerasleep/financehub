# 開始使用

這一節帶你從**零**到**可以登入並建立第一個帳戶**。

| 章節 | 適合對象 | 預估時間 |
| ---- | -------- | -------- |
| [安裝與環境準備](installation.md) | 第一次拿到原始碼的人 | 20 分鐘 |
| [5 分鐘快速上手](quickstart.md) | 已裝好環境想試跑 | 5 分鐘 |

!!! tip "完全不懂技術？"
    現階段系統仍在開發中，需要一些基本指令操作（terminal、git、Docker）。
    待 Sprint 7 部署到 Render 後，使用者只需開瀏覽器即可使用，不需要任何安裝。
